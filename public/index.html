Main features

Libre Radio is a software suite for radio stations, currently under development. Its goal is to provide a modern broadcasting station solutions to many its problems, including automated playout with flexible scheduling and a shared music database that is easy and quick to search while still holding lots of metadata about the recordings that can be useful when searching sore something very specific.

The project is currently under development and will not be ready for production use for quite some time. You can follow our progress on GitHub.

Flexibility

The software suite will be composed of small parts doing simple tasks and can be combined into flexible system fitting mosts situations. Every radio station is different and we don't strive to fit only one.

Ease of use

By splitting a system as complex as a radio station backbone into simpler parts, your producers don't have to be system administrators and can focus on producing high quality programming instead.

Open source

Everything in the software suite is open source and free software, which means you can run the programs however you wish, study how the programs works, modify them and distribute copies to help others.

Getting involved

Want to help make Libre Radio better? All parts of the software suite are licensed under open source licenses.

Coding

The programs making up Libre Radio are written python and C++. If you have experience in any of those languages and you want to contribute, you are more than welcome! Information on how to get started can be found in the GitHub repository for each project.

Translation

The translation work for all projects is done on Transifex.

  1. Go to transifex.com and create an account.
  2. You have to join one of our teams to be able to start translating. Do so by chosing your language from the list and click “Join team”.
  3. We will accept your request as quickly as possible, and in the meantime, you can read up on our translation guidelines.
