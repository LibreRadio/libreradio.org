Install Libre Radio database

Download the source code

You can either download the source code from GitHub or do the same using Git, git clone https://github.com/LibreRadio/database.git.

Run the setup script

Navigate to the /source directory you downloaded. If you'd like to change language, you can run python main.py -l ## where ## is the language code you want to change to.

Run the setup script by typing python main.py --setup. You will then be prompted for database host, username and passord.

The setup script will now create the database structure and save all settings in a configuration file.

Start importing your audio files
