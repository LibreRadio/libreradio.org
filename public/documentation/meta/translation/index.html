Translation

Get started

The translation work for all projects is done on Transifex.

  1. Go to transifex.com and create an account.
  2. You have to join one of our teams to be able to start translating. Do so by chosing your language from the list and click “Join team”.
  3. We will accept your request as quickly as possible, and in the meantime, you can read up on the guidelines below.

Translation guidelines

Variables

Variables inside translatable strings will be a descriptor inside curly brackets, like Please enter a number between {min} and {max}.. The text inside the brackets should never be translated as they will be replaced by something produced by the program.

The variables can be moved around freely inside the string to fit you language's word order.

Developer's checklist

New domain

This is what you will have to do when you add a source file containing translations:

  1. Add the correct domain parameter to the source file, like _ = i18n.i18n("domain_name", settings.get_language())._ for Python.
  2. Add the new file and its domain to the script handling string extraction, and run it. Make sure the script created a new POT file contiaining your strings.
  3. Log in to Tansifex, navigate to the correct project and click on “Resources”.
  4. Click on “Add resources” in the top right corner, select your newly created POT file. The resource should have the same name as the POT file, but without the file extension.
  5. Now, add the domain to the scripts responsible for fetchling the strings from Transifex and processing them to be readable by the code.
  6. Run both those scripts and make sure the output files end up in the correct directories and look as you would expect.

New file

This is what you will have to do when you add a source file containing translatable strings:

  1. Add the code responsible for fetching the internationalized strings from the desired domain and in the right locale. If you are adding this file to a brand new domain, please follow the guide above instead.
  2. Add the new file to the right domain in the script handling string extraction, and run it. Make sure your strings were added to the POT file.
  3. If you have the required permissions, you can update the resource on Transifex to include your new strings.

New language

This is what you will have to do when someone requests a new language on Transifex:

  1. Add the language to the correct project.
  2. Accept the user's request to join the team.
  3. Add the new language code to the script that fetch translations from Transifex.
  4. Add the new language code to the scripts that processes the strings into a format readable by the code.
